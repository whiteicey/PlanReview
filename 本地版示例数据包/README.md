# 本地版示例数据包

> 示例数据 / DEMO_ONLY / 仅用于系统开发测试 / 不作为正式审查依据

## 内容

- `plans/`：4组DOCX方案样本，其中多个样本各包含多个相互独立的问题；
- `standards/`：示例规范条款；
- `historical_opinions/`：Excel/Word历史意见；
- `rules/`：YAML规则和术语映射；
- `golden/`：金标准测试案例。

## 运行预期

- DEMO-001：完整性和主要规则通过；
- DEMO-002：至少发现井数、产能/处理能力、建设周期、投产时间和单位/口径等多类问题；
- DEMO-003：至少发现参数变化、处理能力、建设周期、投产时间、历史意见未闭环和生产井数冲突；
- DEMO-004：至少发现汇总关系、产能计算、能力边界、时间关系、术语和意见回复等多类问题。

## 规则

示例条款、参数和规则均为虚构演示内容，不得作为正式审查依据。
